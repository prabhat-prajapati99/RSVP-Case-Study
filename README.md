# RSVP
This project involves analyzing the IMDB dataset using SQL to uncover meaningful insights about movies, ratings, revenue, genres, directors, and actors. First, the database structure was explored to understand relationships between tables. SQL queries such as SELECT, JOIN, WHERE, GROUP BY, and aggregate functions were used to extract and analyze data. The study identified top-rated and highest-grossing movies, evaluated genre performance, and examined year-wise trends in movie releases. It also highlighted influential directors and actors based on ratings and frequency of releases. Overall, the project demonstrates how SQL can be used to transform raw movie data into actionable business insights........
